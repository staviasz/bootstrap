O que é Bootstrap

Um framework front-end e de código-aberto foi inicialmente criado por Mark Otto e Jacob Thornton para o desenvolvimento web mais rápido e prático. Ele contém todos os tipos de templates baseados em HTML e CSS para várias funções e componentes. Por exemplo, navegação, sistema de grades, carrosséis de imagens e botões.
